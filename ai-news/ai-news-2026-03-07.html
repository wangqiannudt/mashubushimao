1️⃣ 🔥 Cursor 发布 Automations 自动化系统

发布时间：2026年3月5日

2️⃣ 🎤 Claude Code 推出语音模式（Voice Mode）

发布时间：2026年3月3日

核心内容：Anthropic 为 Claude Code 上线语音输入功能。开发者只需按住空格键说话，即可用自然语言描述编程任务。

  • 目前向约5%用户灰度，未来几周逐步开放
  • 支持语音与键盘混合输入，保持流畅体验
  • 大幅减轻RSI（重复性劳损），提升无障碍体验
  • 用户反馈："30分钟语音会话感觉像做了免费按摩"

来源：Deeper Insights
